🔵 向量搜索路径

  • Embedding 编码为 3584 维向量
  • Qdrant 余弦相似度检索
  • 擅长语义理解（"轻便的"→ 相似描述）
  • Payload 索引支持按店铺过滤

🟡 全文搜索路径

  • AI 扩展同义词和关键词
  • MySQL FULLTEXT + ngram 分词
  • 擅长精确匹配（型号 "HX-7SP"）
  • Boolean Mode 通配符搜索
