# ProductSelector - 选品监控工具

## 功能
- 1688热销商品抓取和展示
- 价格/销量/评价趋势监控
- AI分析推荐潜力商品

## 技术栈
- 后端：.NET 8 + ASP.NET Core Web API
- 前端：Vue 3 + Element Plus
- 数据库：MySQL
- 爬虫：HttpClient + 抓取策略
- AI：接入AI意图解析模块

## 启动方式

### 后端
```bash
cd Backend
dotnet restore
dotnet run
```

### 前端
```bash
cd Frontend
npm install
npm run dev
```
